# MockAI

MockAI is currently a frontend-only prototype for local presentation and team review.

## Repository Layout

```text
MockAI/
├── frontend/
├── backend/
├── database/
├── docs/
└── README.md
```

## Frontend Entry Point

Open [frontend/index.html](frontend/index.html) with Live Server in VS Code.

## Run Locally in VS Code

1. Open the `MOKAI-prjt` folder in VS Code.
2. Install the **Live Server** extension by Ritwick Dey if needed.
3. Right-click [frontend/index.html](frontend/index.html) and choose **Open with Live Server**.
4. The app will open in your browser and you can demo every page locally.

## Current Status

- Frontend is ready for local testing.
- Backend, database, and API integration are intentionally not added yet.
- All navigation, scores, and history data are mock data.
- The repository is prepared for a future GitHub push and later Vercel deployment.
